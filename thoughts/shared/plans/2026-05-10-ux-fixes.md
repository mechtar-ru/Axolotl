# UX Fix Plan — Based on Design Retrospective

## Priority P0 (Blocks core UX)

### P0.1 Fix ToastContainer
- **Files:** `frontend/src/App.vue`, `frontend/src/components/ui/ToastContainer.vue`
- **What:** Find where `useToast` state is created. Add `<ToastContainer />` component to `App.vue` template. Verify toasts render.
- **Why:** Toasts are fired everywhere but never visible. This is the single highest-impact fix.

### P0.2 Add Loading States for Schema List
- **Files:** `frontend/src/App.vue`, `frontend/src/stores/schemaStore.ts`
- **What:** Add loading skeleton/spinner for the schema list in the sidebar. Use `schemaStore.loading` (already exists but unused).
- **Why:** Sidebar stays blank during fetch, indistinguishable from failure.

## Priority P1 (High user impact)

### P1.1 Post-Onboarding Coachmarks
- **Files:** `frontend/src/components/ui/OnboardingModal.vue`, new file or composable
- **What:** After wizard closes, overlay 3 highlighted regions with tooltips: (1) +Add button "Add nodes here", (2) Run button "Run when ready", (3) Edge handle "Connect nodes by dragging from port to port"
- **Why:** Most users hit the canvas and don't know what to do next.

### P1.2 Auto-Save Indicator Pill
- **Files:** `frontend/src/components/canvas/WorkflowCanvas.vue`
- **What:** Add a small pill/chip next to schema name showing "Saving...", "Saved", or "Unsaved". Wire auto-save debounce to visual state.
- **Why:** Reliable auto-save with no indicator feels like no auto-save.

## Priority P2 (Visual polish)

### P2.1 Unify Accent Color
- **Files:** `frontend/src/components/ui/TemplateGallery.vue`, `frontend/src/components/panels/RightPanel.vue`
- **What:** Replace `#00bcd4` (cyan) with `#6c63ff` or `--accent` CSS variable in TemplateGallery and RightPanel.
- **Why:** Two accent colors feel like two different products.

### P2.2 Fix Execution Mode Labels
- **Files:** `frontend/src/components/canvas/WorkflowCanvas.vue`
- **What:** Align button text with dropdown options. If dropdown says "Dry Run", button should say "Dry Run" not "Simulate".
- **Why:** Mismatched labels 50px apart erode trust.

## Priority P3 (Feature additions)

### P3.1 Tool Selector on AgentNode
- **Files:** `frontend/src/components/nodes/AgentNode.vue`
- **What:** Add a tool checklist panel to AgentNode. Each tool has name, description, enable/disable toggle. Fetch available tools from backend API.
- **Why:** The #1 gap between README promises and UI reality.

### P3.2 Consolidate Template Systems
- **Files:** `frontend/src/views/HomeView.vue`, `frontend/src/components/ui/TemplateGallery.vue`
- **What:** Either make onboarding fetch from the same API as TemplateGallery, or remove the ambiguity by clarifying their different purposes.
- **Why:** Two template systems with no hierarchy confuse users.
