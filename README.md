# Axolotl — визуальный конструктор AI-агентов

## 🎯 О проекте

Axolotl — это веб-приложение для визуальной оркестрации AI-агентов с использованием node-based графа. 
Позволяет создавать цепочки из источников данных, AI-агентов и выходных узлов, а также импортировать/экспортировать схемы в формате Mermaid.

## ✨ Возможности

- 🎨 Визуальный редактор на основе React Flow
- 🤖 Создание и настройка AI-агентов
- 📥 Источники данных (текст, файлы, базы данных)
- 📤 Узлы вывода результатов
- 💾 Сохранение и загрузка схем в SQLite
- 📊 Экспорт диаграмм в Mermaid
- 📥 Импорт схем из Mermaid
- 🔄 WebSocket для стриминга прогресса выполнения

## 🛠️ Технологии

### Бэкенд
- Java 21
- Spring Boot 3.2
- SQLite
- WebSocket

### Фронтенд
- React 18
- TypeScript
- React Flow
- Axios

## 🚀 Быстрый старт

### Запуск бэкенда
```bash
mvn spring-boot:run
```

### Запуск фронтенда
```bash
cd frontend
npm install
npm start
```
