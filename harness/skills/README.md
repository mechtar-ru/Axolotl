# Skills

Skills are reusable instruction modules loaded by opencode agents.

## Available Skills

| Skill | File | Purpose |
|-------|------|---------|
| Playwright CLI | `playwright-cli` | Run Playwright tests, capture HTML snapshots |

## Creating a Skill

1. Create a directory under `harness/skills/<name>/`
2. Write SKILL.md with instructions
3. Reference it from sub-agent prompts
