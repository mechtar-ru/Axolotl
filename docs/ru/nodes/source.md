# Узел Source

Узел **Source** — это точка входа для данных в ваш workflow. Он предоставляет начальные данные, которые обрабатывают последующие узлы.

## Обзор

```
┌─────────────────┐
│      Source     │
│                 │
│  ┌───────────┐  │
│  │ Входной   │  │
│  │   текст   │  │
│  └───────────┘  │
│                 │
│      🔽 Выход   │
└─────────────────┘
```

## Настройка

| Свойство | Тип | Описание |
|----------|------|----------|
| **name** | string | Имя узла (двойной клик для редактирования) |
| **sourceData** | string | Входной текст или данные |

## Использование

1. Добавьте узел Source с панели инструментов
2. Двойной клик для раскрытия
3. Введите текст в textarea
4. Соедините выход с входом другого узла

## Пример

```javascript
// Содержимое узла Source:
"Создай план для React приложения"

// Подключен к узлу Agent → "Создать подробный план"
// Подключен к узлу Output → Показать результаты
```

## Drag & Drop файлов

Вы можете перетащить текстовый файл на узел Source для загрузки его содержимого.

## Следующий узел

Выход Source обычно подключается к:

- [Condition](/ru/nodes/condition) - Ветвление по содержимому
- [Memory](/ru/nodes/memory) - Сохранение в MemPalace
