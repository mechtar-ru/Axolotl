# Узел Agent

Узел **Agent** — это ядро Axolotl. Он вызывает Large Language Models для обработки и трансформации данных.

## Обзор

```
┌─────────────────┐
│      Agent      │
│    🤖  GPT-4   │
│  ┌───────────┐  │
│  │  Промпт   │  │
│  └───────────┘  │
│ 🔽 Выход        │
└─────────────────┘
```

## Настройка

| Свойство | Тип | Описание |
|----------|------|----------|
| **name** | string | Имя узла |
| **userPrompt** | string | Шаблон пользовательского промпта |
| **model** | string | Выбор LLM модели |
| **systemPrompt** | string | (опционально) Системные инструкции |

## Поддерживаемые провайдеры

| Провайдер | Модели |
|------------|--------|
| **OpenAI** | GPT-4o, GPT-4o-mini, GPT-4 Turbo |
| **Anthropic** | Claude Sonnet, Claude Opus, Claude Haiku |
| **Ollama** | Любая локальная модель (llama3, mistral, и т.д.) |
| **DeepSeek** | DeepSeek Chat |
| **Cloudflare** | Workers AI (kimi-k2.6, и т.д.) |
| **Custom** | Любой OpenAI-совместимый эндпоинт |

## Интерполяция переменных

Используйте `{{переменные}}` в промптах:

```javascript
// Пример промпта с переменными:
"{{input}}"

"Сократи это: {{input}}"

"Ты код-ревьюер. Проверь этот код:\n{{input}}\n\nФокус на: {{focus}}"
```

Доступные переменные:
- `{{input}}` - Выход предыдущего узла
- `{{prev_result}}` - Последний результат
- `{{node:Имя}}` - Выход конкретного узла

## Стриминг

Включите посимвольный вывод для отображения LLM ответа в реальном времени.

## Использование

1. Добавьте узел Agent
2. Подключите вход от Source или другого узла
3. Двойной клик для открытия редактора промпта
4. Выберите модель из списка
5. Напишите промпт (используйте `{{input}}` для результата)
6. Выполните workflow
