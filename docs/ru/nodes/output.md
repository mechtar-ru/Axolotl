# Узел Output

Узел **Output** отображает финальные результаты выполнения вашего workflow.

## Обзор

```
┌─────────────────┐
│      Output     │
│                 │
│  Область        │
│  результатов    │
│                 │
│ 🔽 Финальный    │
│    вывод        │
└─────────────────┘
```

## Настройка

| Свойство | Тип | Описание |
|----------|------|----------|
| **name** | string | Имя узла |
| **template** | string | (опционально) Шаблон вывода |

## Использование

Узел Output собирает данные из:
- Прямых подключений от любого узла
- Ссылки на переменные `{{node:Agent}}`

## Пример

```javascript
// Простой вывод
{{input}}

// Форматированный вывод
# Результаты

{{input}}

---
Сгенерировано Axolotl

// Мульти-источник
Вход: {{input}}

Резюме: {{node:Резюме}}
Код: {{node:Генератор}}
```

## Форматы вывода

Узел Output поддерживает:
- Простой текст
- Markdown (рендерится)
- JSON (форматированный)
- Агрегация нескольких переменных

## Конец workflow

Узел Output обычно является финальным узлом workflow, завершая поток данных.

## Связанные узлы

- [Source](/ru/nodes/source) - Ввод данных
- [Agent](/ru/nodes/agent) - Обработка данных
