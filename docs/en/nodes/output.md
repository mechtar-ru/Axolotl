# Output Node

The **Output** node displays the final results of your workflow execution.

## Overview

```
┌─────────────────┐
│      Output     │
│                 │
│  Results area   │
│                 │
│ 🔽 Final output │
└─────────────────┘
```

## Configuration

| Property | Type | Description |
|----------|------|-------------|
| **name** | string | Node name |
| **template** | string | (optional) Output template |

## Usage

The Output node collects data from:
- Direct connections from any node
- Variable references like `{{node:Agent}}`

## Example

```javascript
// Simple output
{{input}}

// Formatted output
# Results

{{input}}

---
Generated by Axolotl

// Multi-source output
Input: {{input}}

Summary: {{node:Summary}}
Code: {{node:CodeGenerator}}
```

## Output Formats

The Output node supports:
- Plain text
- Markdown (rendered)
- JSON (formatted)
- Multi-variable aggregation

## Workflow End

The Output node is typically the final node in a workflow, marking the end of data flow.
