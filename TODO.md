## ✨ Новая функциональность

### Фронтенд

#### Узлы (Nodes)
- [ ] **Condition Node** — узел с условиями (if/else) для ветвления логики
- [ ] **Loop Node** — узел для циклического выполнения подграфа
- [ ] **Memory Node** — узел для хранения контекста между выполнениями
- [ ] **Guardrail Node** — валидация выходов агента (безопасность, качество)
- [ ] **Fallback Node** — обработка ошибок и retry-механизмы
- [ ] **Human Node** — ожидание подтверждения пользователя (human-in-the-loop)
- [ ] **Webhook Node** — запуск схемы по HTTP-вызову
- [ ] **Schedule Node** — запуск по расписанию (cron)

#### Редактор
- [ ] **Группировка узлов** — создание подграфов для сложных схем
- [ ] **Комментирование** — добавление текстовых заметок на канвас
- [ ] **Цветовая индикация типов** — Source (зелёный), Agent (фиолетовый), Output (оранжевый), Condition (синий)
- [ ] **Поиск по схеме** — поиск узлов по имени или типу
- [ ] **Сворачивание узлов** — возможность минимизировать узел для экономии места
- [ ] **Drag & drop загрузка файлов** — перетаскивание файлов в SourceNode

#### Интерфейс
- [ ] **Редактирование промптов в отдельном окне** — полноэкранный редактор с подсветкой синтаксиса
- [ ] **Сохранение истории чатов** — сообщения между пользователем и агентом
- [ ] **Вкладка конфигурации** — настройка подключений к LLM (OpenClaw, Ollama, OpenAI)
- [ ] **Проверка токенов** — отображение остатка для API моделей
- [ ] **Темы оформления** — светлая/тёмная тема
- [ ] **Локализация** — поддержка нескольких языков
